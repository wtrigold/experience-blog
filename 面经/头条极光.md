## 11.双向链表反转

```js
function reverse(head){
    let temp=null
    let current=head
    while(current){
        temp=current.pre
        current.pre=current.next
        current.next=temp
        current=current.prev
    }
    if(temp!==null){
        head=temp.prev
    }
}
```
